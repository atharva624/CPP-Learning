# Stack-Based Expression Evaluator

## 1. Problem Definition:
The project involves developing a stack-based application with two primary functions:
1. Evaluating mathematical expressions given in postfix notation.

It requires handling input/output, utilizing classes and the standard library, and implementing functions for modular code.

## 2. Solution:
1. **ExpressionEvaluator Class:** This class encapsulates the functionality for evaluating postfix expressions and reversing strings.
2. **Postfix Evaluation:** The `evaluatePostfix` function processes the input string and uses a stack to compute the result of the expression.
3. **Main Function:** The `main` function creates an object of `ExpressionEvaluator` and demonstrates the usage of both functionalities.

## 3. Learning:
- **Stacks:** Demonstrates the application of stacks in evaluating expressions and manipulating strings.
- **Object-Oriented Programming (OOP):** Uses classes to encapsulate related functionalities, promoting modular design.
- **Standard Library:** Utilizes the `<stack>`, `<string>`, and `<sstream>` libraries for efficient data handling.
- **Input/Output Handling:** Shows how to handle user input and output with `cin` and `cout`.
